Une assemblée de "n" personnes a voté à bulletin secret pour élire son président parmi plusieurs candidats (par exemple Jacques, Michel, Pierre et Olivier). Concevoir l'algorithme qui traite le dépouillement de ce vote et qui fournit le résultat. Entrées : Tableau $candidats des candidats. Tableau $votes des noms inscrits sur tous les bulletins dépouillés. Par exemple : $candidats = ["Jacques", "Michel", "Pierre", "Olivier"]; $votes = ["Jacques", "Pierre", "Jacques", "Michel", "Pierre", "Jacques", "Jacques", "Jacques", "Jacques", "Jacques", "Jacques", "Jacques", "Jacques"]; Sorties : Message 1 "Le candidat ... a obtenu ... votes sur ..., soit ... %." Message 2 "Le candidat ... est élu." Message 3 "Les candidats ... ont obtenu le même score, un nouveau vote doit être organisé avec ces candidats. Étapes : Étape 1 : Comptabiliser le nombre de bulletins obtenu par chaque candidat. Étape 2 : Pour chaque candidat afficher le message 1. Étape 3 : Déterminer le meilleur score et le(s) candidat(s) qui a(ont) obtenu ce score. Étape 4 : Si un seul candidat a obtenu le meilleur score, afficher le message 2, sinon afficher le message 3.
